# Entrepreneurship Master Quiz

A web-based quiz application to help students master entrepreneurship concepts.

## Features
- **Learn by Topic**: Go through questions topic by topic.
- **Random Quiz**: Test your knowledge with shuffled questions.
- **Flashcards**: Review questions based on mastery (New, Correct, Wrong).
- **PWA Support**: Installable on iOS devices as a native-like app.

## How to Install on iOS
1. Open the website in Safari on your iPhone or iPad.
2. Tap the "Share" button (rectangle with an arrow pointing up).
3. Scroll down and tap "Add to Home Screen".
4. Tap "Add".
